# Rickpedia API

Projeto backend desenvolvido com Django e Django REST Framework que consome a API pública de Rick and Morty, permitindo gerenciar personagens, favoritos e listas personalizadas.

---

## 🚀 Tecnologias utilizadas

- Python
- Django
- Django REST Framework
- PostgreSQL
- Docker
- Swagger (drf-yasg)

---

## 📦 Funcionalidades

- CRUD completo de listas
- Sistema de favoritos
- Integração com API externa (Rick and Morty)
- Autenticação com Token
- Relacionamento entre entidades (User, List, Character)
- Documentação com Swagger
